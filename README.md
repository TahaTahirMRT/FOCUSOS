## 🧠 Overview

FocusOS is a modern productivity web application designed to help users improve focus, reduce distractions, and build deep work habits.

In today’s digital world, attention span is shrinking due to social media, notifications, and constant interruptions. FocusOS solves this by tracking user focus behavior and providing actionable insights to improve productivity.

---

## 🎯 Problem Statement

People today struggle with:
- Short attention spans
- Constant distractions
- Lack of productivity tracking
- Poor time management

FocusOS turns attention into measurable data and helps users improve it.

---

## 🚀 Features

### ⏱ Focus Timer
- Start, pause, and stop focus sessions
- Pomodoro-based sessions (25/45/60 min)
- Automatic session tracking

### 📊 Analytics Dashboard
- Daily focus time tracking
- Weekly performance charts
- Productivity score (0–100)
- Streak tracking system

### 🚫 Distraction Tracking
Users log distractions after each session:
- Social Media
- YouTube
- WhatsApp
- Gaming
- Other

### 🧠 Focus Score System
Calculated based on:
- Completed sessions
- Interruptions
- Consistency
- Streaks

### 🏆 Gamification
- Daily streaks
- Achievement badges
- Level progression system

### 📈 Data Visualization
- Focus trend charts
- Distraction breakdown pie chart
- Weekly progress graphs

## 🛠 Tech Stack

- Next.js (React Framework)
- Tailwind CSS
- Chart.js
- Supabase (Auth + Database)
- Vercel (Deployment)

---

## 🗄 Database Schema

Tables used:
- users (Supabase Auth)
- focus_sessions
- distractions
- user_stats
- achievements
